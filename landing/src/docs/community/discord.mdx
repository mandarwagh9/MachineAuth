# Discord

Join our community on Discord for help, discussions, and updates.

## Join Link

**[Join MachineAuth Discord](https://discord.gg/machineauth)**

## Channels

| Channel | Description |
|---------|-------------|
| `#general` | General discussion |
| `#help` | Get help with MachineAuth |
| `#showcase` | Share what you built |
| `#announcements` | Release updates |
| `#feature-requests` | Suggest new features |

## Community Guidelines

1. **Be respectful** — Treat everyone with respect
2. **Stay on topic** — Keep discussions relevant
3. **No spam** — Don't promote products without permission
4. **Help others** — Share your knowledge

## Getting Help

1. Check the [docs](/docs/getting-started/introduction)
2. Search existing questions
3. Ask in `#help`
4. Open a GitHub issue for bugs

## Contributing

Want to help the community?
- Answer questions in `#help`
- Share your projects in `#showcase`
- Report bugs on GitHub
